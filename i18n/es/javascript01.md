### Open console

Since you are now an aspiring programmer üñ•Ô∏è, you can use it too! To bring up the console, simply press the üîò `CTRL, SHIFT (or COMMAND and OPTION if you have a Mac) and J` keys together.

## Hello world program

"Hello, world!" - A program that outputs the phrase "Hello, world!" To the screen or other device üåé

This is usually the first example of a program in a programming textbook, and for many students, this is the first experience in learning a new languageüëÖ.

Now that the console window is open in front of you and you have stretched it in a convenient way, click the mouse to the right of the angle bracket `>`.

![not equal](https://media.giphy.com/media/jPfQcPdmI9bTXpa7hi/giphy.gif)

A blinking cursorüñ±Ô∏è appears on the line, indicating that you can start typing. You will now type your first lines of codeüìü. Enter the following text üìú (including quotation marks):

```
"Hello world!"
```

![Hello world!](/img/javascript/03.png)
![Hello World](https://media.giphy.com/media/xiOgHgY2ceKhm46cAj/giphy.gif)

Press ENTER. The console will return your text on the next line üìú: "Hello World!". Through the browser console, we communicate directly with the JavaScript engine - this is such a program, or, in other words, an interpreter that executes code written in JavaScript.

![Hello world!](/img/javascript/04.png)

Now you can be proud that you wrote and executed the first JavaScript code with your own hands!

Then press the key\_ UP_ARROW ‚¨ÜÔ∏è (up arrow) and you will see your text üìú "Hello World!". Next, move the cursorüñ±Ô∏è to the beginning of the line and type the following text instead üìú:

```javascript
var hello = 'Hello world!'
```

and press ENTER. In the console you will see the answer: `undefined`. Everything goes according to plan! Now type:

```javascript
hello
```

and press ENTER. If you did everything correctly, the console will return a message in the line üí¨ "Hello, World!". Here we have created a variable and assigned a value to it.

![Hello world!](/img/javascript/05.png)

And, of course, this is the minimum that can be done in the browser console. Further more!
