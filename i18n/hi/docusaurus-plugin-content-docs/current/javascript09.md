## Arrow Function (╤Б╤В╤А╨╡╨╗╨╛╤З╨╜╤Л╨╡ ╤Д╤Г╨╜╨║╤Ж╨╕╨╕)

![Arrow](https://media.giphy.com/media/xT9IgAakXAITtXIWje/giphy.gif)

╨б╤Г╤Й╨╡╤Б╤В╨▓╤Г╨╡╤В ╨╡╤Й╤С ╨▒╨╛╨╗╨╡╨╡ ╨┐╤А╨╛╤Б╤В╨╛╨╣ ╨╕ ╨║╤А╨░╤В╨║╨╕╨╣ ╤Б╨╕╨╜╤В╨░╨║╤Б╨╕╤Б   ╨┤╨╗╤П ╤Б╨╛╨╖╨┤╨░╨╜╨╕╤П ╤Д╤Г╨╜╨║╤Ж╨╕╨╣тЪЩя╕П, ╨║╨╛╤В╨╛╤А╤Л╨╣ ╤З╨░╤Б╤В╨╛ ╨╗╤Г╤З╤И╨╡, ╤З╨╡╨╝ ╨┤╤А╤Г╨│╨╕╨╡ ╨▓╨╕╨┤╤Л.

╨Ш╤В╨░╨║, ╨║╨╗╨░╤Б╤Б╨╕╤З╨╡╤Б╨║╨╕╨╣ ╨▓╨░╤А╨╕╨░╨╜╤В ╨╖╨░╨┐╨╕╤Б╨╕  ╤Д╤Г╨╜╨║╤Ж╨╕╨╕тЪЩя╕П:

Function Declaration:

```javascript
function func1(arg1, arg2, ...argN) {
  return expression
}
```

Function Expression:

```javascript
let func1 = function (arg1, arg2, ...argN) {
  return expression

```

╨в╨░╨║╨╛╨╣ ╨║╨╛╨┤ ╤Б╨╛╨╖╨┤╨░╤С╤В ╤Д╤Г╨╜╨║╤Ж╨╕╤ОтЪЩя╕П `func1` ╤Б ╨░╤А╨│╤Г╨╝╨╡╨╜╤В╨░╨╝╨╕ `arg1, .. argN` ╨╕ ╨▓╤Л╤З╨╕╤Б╨╗╤П╨╡╤В `expression` - ╨╜╨░╨▒╨╛╤А ╨┤╨╡╨╣╤Б╤В╨▓╨╕╨╣ ╨░╨╗╨│╨╛╤А╨╕╤В╨╝╨░ ╤Б ╨┐╤А╨░╨▓╨╛╨╣ ╤Б╤В╨╛╤А╨╛╨╜╤Л ╤Б ╨╕╤Е ╨╕╤Б╨┐╨╛╨╗╤М╨╖╨╛╨▓╨░╨╜╨╕╨╡╨╝, ╨▓╨╛╨╖╨▓╤А╨░╤Й╨░╤П ╤А╨╡╨╖╤Г╨╗╤М╤В╨░╤В ╨▓╤Л╤З╨╕╤Б╨╗╨╡╨╜╨╕╨╣ ╤З╨╡╤А╨╡╨╖ ╨║╨╗╤О╤З╨╡╨▓╨╛╨╡ЁЯЧЭя╕П ╤Б╨╗╨╛╨▓╨╛ `return`.

╨в╨╡╨┐╨╡╤А╤М ╨┤╨░╨▓╨░╨╣╤В╨╡ ╨┐╤А╨╕╨╝╨╡╨╜╨╕╨╝ ╤Б╤В╤А╨╡╨╗╨╛╤З╨╜╤Л╨╡ ╤Д╤Г╨╜╨║╤Ж╨╕╨╕ - `Arrow functions`:


╨Ф╤А╤Г╨│╨╕╨╝╨╕ ╤Б╨╗╨╛╨▓╨░╨╝╨╕, ╤Н╤В╨╛ ╨▒╨╛╨╗╨╡╨╡ ╨║╨╛╤А╨╛╤В╨║╨╕╨╣ ╨▓╨░╤А╨╕╨░╨╜╤В ╤В╨░╨║╨╛╨╣ ╨╖╨░╨┐╨╕╤Б╨╕  .

:::note ╨Т╨╛ ╨║╨░╨║ !
`=>` ╨╜╨░╨╝ ╨╖╨░╨╝╨╡╨╜╨╕╨╗╨░ `{ return ... }` ╤Б ╨┐╤А╨░╨▓╨╛╨╣ ╤Б╤В╨╛╤А╨╛╨╜╤Л ╨╕ ╨┐╨╛╨╖╨▓╨╛╨╗╨╕╨╗╨░ ╨╜╨╡ ╨┐╨╕╤Б╨░╤В╤М ╨║╨╗╤О╤З╨╡╨▓╨╛╨╡ ЁЯЧЭя╕П ╤Б╨╗╨╛╨▓╨╛ `function` ╤Б ╨╗╨╡╨▓╨╛╨╣ ╤Б╤В╨╛╤А╨╛╨╜╤Л!
:::

╨Ф╨░╨▓╨░╨╣╤В╨╡ ╨▓╨╖╨│╨╗╤П╨╜╨╡╨╝ ╨╜╨░ ╨║╨╛╨╜╨║╤А╨╡╤В╨╜╤Л╨╣ ╨┐╤А╨╕╨╝╨╡╤А  :

```jsx live

  // ╨Ш╤Б╤Е╨╛╨┤╨╜╨░╤П ╤Д╨╛╤А╨╝╨░ ╤Д╤Г╨╜╨║╤Ж╨╕╨╕ ╤Б╨╗╨╛╨╢╨╡╨╜╨╕╤П 2-╤Е ╤З╨╕╤Б╨╡╨╗
```
