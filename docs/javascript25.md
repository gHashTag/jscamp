–° –∞–Ω–≥–ª–∏–π—Å–∫–æ–≥–æ `loop` –ø–µ—Ä–µ–≤–æ–¥–∏—Ç—Å—è –∫–∞–∫ "–ø–µ—Ç–ª—è", —á—Ç–æ –æ—Ç–ª–∏—á–Ω–æ –æ—Ç—Ä–∞–∂–∞–µ—Ç —Å–º—ã—Å–ª: –º—ã –∏–º–µ–µ–º –¥–µ–ª–æ —Å –∑–∞–∫–æ–ª—å—Ü–æ–≤–∞–Ω–Ω–æ–π –æ—á–µ—Ä–µ–¥—å—é.

`Event loop` —Ä–µ–≥—É–ª–∏—Ä—É–µ—Ç –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –∏—Å–ø–æ–ª–Ω–µ–Ω–∏—è –∫–æ–Ω—Ç–µ–∫—Å—Ç–æ–≤ ‚Äî —Å—Ç–µ–∫. –û–Ω —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç—Å—è, –∫–æ–≥–¥–∞ —Å—Ä–∞–±–æ—Ç–∞–ª–æ —Å–æ–±—ã—Ç–∏–µ –∏–ª–∏ –±—ã–ª–∞ –≤—ã–∑–≤–∞–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è‚öôÔ∏è. –†–µ–∞–∫—Ü–∏—è –Ω–∞ —Å–æ–±—ã—Ç–∏–µ –ø–æ–º–µ—â–∞–µ—Ç—Å—è –≤ –æ—á–µ—Ä–µ–¥—å –∏—Å–ø–æ–ª–Ω–µ–Ω–∏—è, –≤ `event loop`, –∫–æ—Ç–æ—Ä—ã–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, —Å –∫–∞–∂–¥—ã–º —Ü–∏–∫–ª–æ–º –≤—ã–ø–æ–ª–Ω—è–µ—Ç –ø–æ–ø–∞–¥–∞—é—â–∏–π –≤ –Ω–µ–≥–æ –∫–æ–¥üìü . –ü—Ä–∏ —ç—Ç–æ–º –ø—Ä–∏–≤—è–∑–∞–Ω–Ω–∞—è –∫ —Å–æ–±—ã—Ç–∏—é —Ñ—É–Ω–∫—Ü–∏—è‚öôÔ∏è –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è —Å–ª–µ–¥—É—é—â–µ–π –ø–æ—Å–ª–µ —Ç–µ–∫—É—â–µ–≥–æ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞ –∏—Å–ø–æ–ª–Ω–µ–Ω–∏—è.

–í JavaScript –ø–æ—Å—Ç–æ—è–Ω–Ω–æ —Ä–∞–±–æ—Ç–∞—é—Ç —Å–≤—è–∑–∞–Ω–Ω—ã–µ –º–µ–∂–¥—É —Å–æ–±–æ–π —Å–∏–Ω—Ö—Ä–æ–Ω–Ω–∞—è –∏ –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–∞—è –æ—á–µ—Ä–µ–¥–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è. –°–∏–Ω—Ö—Ä–æ–Ω–Ω–∞—è ‚Äî `stack` ‚Äî —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç –æ—á–µ—Ä–µ–¥—å –∏ –ø—Ä–æ–±—Ä–∞—Å—ã–≤–∞–µ—Ç –≤ –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—É—é ‚Äî `event loop` ‚Äî –≤—ã–∑–æ–≤—ã —Ñ—É–Ω–∫—Ü–∏–π‚öôÔ∏è, –∫–æ—Ç–æ—Ä—ã–µ –±—É–¥—É—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω—ã –ø–æ—Å–ª–µ —Ç–µ–∫—É—â–µ–≥–æ –∑–∞–ø–ª–∞–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –∏—Å–ø–æ–ª–Ω—è–µ–º–æ–≥–æ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞.

## –û–ø–∏—Å–∞–Ω–∏–µ

–ö —Å—á–∞—Å—Ç—å—é, –±—Ä–∞—É–∑–µ—Ä –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –Ω–∞–º –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ —Å–∞–º –º–µ—Ö–∞–Ω–∏–∑–º JavaScript –Ω–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç: Web API. –ö–æ—Ç–æ—Ä—ã–π –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è DOM API, setTimeout, HTTP-–∑–∞–ø—Ä–æ—Å—ã –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ. –≠—Ç–æ –º–æ–∂–µ—Ç –ø–æ–º–æ—á—å –Ω–∞–º —Å–æ–∑–¥–∞—Ç—å –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ–µ –Ω–µ–±–ª–æ–∫–∏—Ä—É—é—â–µ–µ –ø–æ–≤–µ–¥–µ–Ω–∏–µ üöÄ.
